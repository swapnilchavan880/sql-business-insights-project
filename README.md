# 📊 SQL Business Insights Project

## 🔍 Overview
This project focuses on solving real-world business problems using SQL. Based on a set of ad-hoc business requests, I extracted actionable insights to support strategic decision-making for top-level management.

## 🎯 Objective
- Analyze business data using SQL
- Answer 10 ad-hoc business questions
- Deliver insights in a clear and structured format
- Present findings to stakeholders using visual storytelling

## 🛠️ Tools & Technologies
- MySQL – Data querying and analysis  
- Excel – Data validation and visualization  
- PowerPoint – Business presentation and storytelling  

## 📁 Project Structure
- `/SQL Queries` → Contains all SQL scripts for ad-hoc requests  
- `/Dataset` → Raw data used for analysis  
- `/Excel` → Processed data and charts  
- `/Presentation` → Final business presentation (PPT)  
- `ad-hoc-requests.pdf` → Business problem statements  

## 📈 Key Highlights
- Solved 10 real-world business queries using SQL  
- Used joins, aggregations, CTEs, and window functions  
- Identified trends, top-performing products, and sales insights  
- Created management-level presentation with clear insights
